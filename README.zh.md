## 🚀 Spring AI 实战项目（Spring AI Playground）

这是一个基于 **Spring AI** 构建的实战型项目，目标是帮助 Java 开发者：

👉 学会 Spring AI
👉 理解 Agent 架构
👉 有信心用 Java 构建 AI 应用

这不仅是一个 Demo，更是一个 **学习指南 + 能力展示项目**。

---

## ✨ 为什么做这个项目？

很多 Java 开发者在 AI 面前会有这些顾虑：

* “AI 是 Python 的天下”
* “Agent 太复杂，看不懂”
* “不知道从哪里开始”

这个项目就是来解决这些问题的。

👉 用你熟悉的 Spring 技术栈，一样可以做 AI。

---

## 🧠 你可以学到什么？

* Spring AI 的核心用法
* 如何构建 Agent 工作流
* Prompt Engineering（结构化实现）
* Function Calling / Tool Calling
* 上下文记忆（Memory）管理
* RAG（检索增强生成）
* AI 后端服务设计思路

---

## 🔧 技术栈

* Java 17+
* Spring Boot
* Spring AI
* OpenAI / 其他大模型
* 向量数据库（Redis / PGVector / Milvus）
* Maven / Gradle

---

## 🧩 核心功能

* ✅ LLM 对话能力（Chat）
* ✅ Agent 流程编排
* ✅ 工具调用（让 AI 执行代码能力）
* ✅ RAG 知识库问答
* ✅ 模块化设计，易扩展
* ✅ 面向生产的架构设计

---

## 🎯 项目目标

1. **帮助学习**
   让开发者快速上手 Spring AI

   证明 Java 一样可以做 AI Agent

3. **展示能力**
   用真实场景体现 AI 应用架构

## 📂 项目结构

```
├── agent/              # Agent 编排逻辑
├── chat/               # 对话与模型调用
├── rag/                # 检索增强生成
├── tools/              # 工具调用实现
├── config/             # Spring AI 配置
└── examples/           # 示例与用例
```
